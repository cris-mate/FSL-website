# Matei Manuela Natalis — FSL

## Requisiti
- XAMPP (Apache + MySQL)
- PHP 7.4+

## Installazione

1. Copiare la cartella nella directory `htdocs` di XAMPP
2. Avviare Apache e MySQL da XAMPP Control Panel
3. Aprire phpMyAdmin (`http://localhost/phpmyadmin`)
4. Importare il file `sql/database.sql` (crea database e dati di esempio)
5. Aprire `http://localhost/MateiManuelaNatalis_FSL/`
